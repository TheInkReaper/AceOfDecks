Política de Privacidad de Ace Of Decks

1. Información General
Ace Of Decks es una aplicación de código abierto (FOSS) diseñada para el estudio mediante flashcards. Esta aplicación está diseñada para funcionar de manera local en el dispositivo del usuario.

2. Recopilación de Datos

  * Datos Personales: Ace Of Decks no recopila, no almacena y no comparte ninguna información personal identificable (como nombre, correo electrónico o ubicación).

  * Datos de Uso: No utilizamos herramientas de análisis (analytics) ni rastreadores de terceros.

  * Datos de la Aplicación: Todos los mazos, tarjetas y progresos de estudio se almacenan localmente en la base de datos de tu dispositivo. El desarrollador no tiene acceso a estos datos.

3. Permisos del Dispositivo
La aplicación puede requerir acceso al almacenamiento para las funciones de:

  * Importación de archivos JSON (mazos).

  * Exportación de archivos JSON.
Estos permisos se utilizan exclusivamente para la gestión de tus propios datos.

4. Terceros
Al ser una aplicación sin anuncios y que no requiere conexión a internet para su funcionamiento principal, no se comparten datos con ninguna empresa externa.

5. Contacto
Si tienes preguntas sobre esta política, puedes contactar al desarrollador a través del repositorio oficial en GitHub: https://github.com/TheInkReaper/AceOfDecks
